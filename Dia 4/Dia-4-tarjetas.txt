Operador AND (and)
Este operador devuelve True solo si ambas condiciones que se comparan son verdaderas. Si alguna es falsa, devuelve False.
mi_bool = 10 > 5 and 8 == 8
print(mi_bool)
Resultado de este código:
True

mi_bool = 10 == 5 and 8 == 8
print(mi_bool)
Resultado de este código:
False

Operador OR (or)
Este operador devuelve True si alguna de las condiciones es verdadera. Solo devuelve False si todas son falsas.
mi_bool = 10 == 5 or 8 == 8
print(mi_bool)
Resultado de este código:
True

mi_bool = 10 == 5 or 8 == 7
print(mi_bool)
Resultado de este código:
False

Operador NOT (not)
Este operador invierte el valor de la condición, es decir, si la condición es True devuelve False y si es False devuelve True.
mi_bool = not (10 == 10)
print(mi_bool)
Resultado de este código:
False

mi_bool = not (5 > 10)
print(mi_bool)
Resultado de este código:
True

-------Loop_While------
el buble while se usa para que cuando se cumpla cuna condicion se ejecute un codigo hata que la condicion ya no se cumpla
la estructura de un buble while es
while <condicion>:
    bloque si se cumple la condicion
else:
    bloque hasta que no se cumpla la condicion

ejemplo basico
monedas = 5

while monedas > 0:
    print(f"Tengo {monedas} monedas")
    monedas -= 1
else:
    print("Ya no tengo monedas")

otro ejemplo un poco mas avanzado
vidas = 5

while vidas > 0:
    if  vidas == 1:
        print(f"Tengo {vidas} vida restante")
        vidas-=1
    else:
        print(f"Tengo {vidas} vidas restantes")
        vidas -= 1
else:
    print("Ya no tengo vidas")
resultado de este codigo es:
Tengo 5 vidas restantes
Tengo 4 vidas restantes
Tengo 3 vidas restantes
Tengo 2 vidas restantes
Tengo 1 vida restante
Ya no tengo vidas

otro ejemplo para pregunta si se quiere continuar con el proceso
respuesta = 's'
while respuesta == 's':
    respuesta = input('Quieres continuar? (s/n)')
else:
    print("Gracias")
resultado de este codigo es:
Quieres continuar? (s/n)s
Quieres continuar? (s/n)s
Quieres continuar? (s/n)n
Gracias

-------Enumerador-------
primero veamos un codigo en el cual no usamos enumerate
lista = ['a','b','c']
indice = 0
for item in lista:
    print(indice,item)
    indice += 1

y ahora vemoas una forma ma eficiente de mostrar el mismo resultado pero con menos docito usando el metodo de Enumerate
lista = ['a','b','c']
for indice, item in enumerate(lista):
    print(indice,item)

ambos muestra este mismo resultado
0 a
1 b
2 c

tambien el metodo enumerate lo puede hacer si agregar dos variables
lista = ['a','b','c']
for item in enumerate(lista):
    print(item)
Enumerate se encarga de enumararlas automaticamente si solo especificamos una variable
este es el resultado del codigo:
(0, 'a')
(1, 'b')
(2, 'c')

tambien podemos poner este resultado de tuples,
tambien podemos enlistar estos tuples que nos devulve el enumerate
lista = ['a','b','c']
mis_tuples = list(enumerate(lista))
print(mis_tuples)
el resultaod de este codigo es:
[(0, 'a'), (1, 'b'), (2, 'c')]


---------zip--------
el zip nos ayuda a que cuando tengamos dos listas
podasmos juntar un valor de una lista y otro valor de otra lista dentro de tuples
nombre = ['Rodrigo', 'Alberto', 'Michael', 'Freddy']
edades= [25, 37, 43]
nueva_lista = list(zip(nombre, edades))
print(nueva_lista)

resultado de este codigo es:
[('Rodrigo', 25), ('Alberto', 37), ('Michael', 43)]

tambine pueden ser dos o mas listas
nombre = ['Rodrigo', 'Alberto', 'Michael', 'Freddy']
edades= [25, 37, 43]
ciudades = ['Madrid', 'Buenos Aires', 'Mixco']
nueva_lista = list(zip(nombre, edades, ciudades))
print(nueva_lista)

una forma de verlo es podemo mostrar estos valores a la vez gracias a zip
nombre = ['Rodrigo', 'Alberto', 'Michael', 'Freddy']
edades= [25, 37, 43]
ciudades = ['Madrid', 'Buenos Aires', 'Mixco']
nueva_lista = list(zip(nombre, edades, ciudades))

for nombre, edades, ciudades in nueva_lista:
    print(f'{nombre} tiene {edades} anios y vive en {ciudades}')
el resultado de este codigo es:
Rodrigo tiene 25 anios y vive en Madrid
Alberto tiene 37 anios y vive en Buenos Aires
Michael tiene 43 anios y vive en Mixco

el importante mencionar que zip se acopla a la cantidad de elemento de la lista mas pequeña es decir si una lista tiene 3 elementos y las demas tiene 4 o mas elementos zip solo va a otmar 3 elementos porque una lista solo tiene 3 elementos

--------Random--------
para utilizar random tenemos que importar una libreria
from random import *

metodo randint(inicio, limite) con dos parametros
este lo que hará es selecionar un numero completamente al azar entre el inicio y el limite impuesto en los paramteos
from random import *
aleatorio = randint(1, 50)
print(aleatorio)

uniform(inicio, limite) numero aleatorio con decimales al azar entre el inicio y limite
from random import *
aleatorio = uniform(1, 2)
print(aleatorio)
tambien podemos conbinarlo con round() con el segundo aprametros para redondear los decimales
from random import *
aleatorio = round(uniform(1, 2),2)
print(aleatorio)

random() sin parametros
random sin parametros devuelve el valor aleatorio entre 0 y 1
from random import *
aleatorio = random()
print(aleatorio)
